# WikiLoop 国际化（i18n）设计文档

## 目标

为 WikiLoop 实现中/英双语支持：
- Web UI（index.html、settings.html、files.html）前端动态切换
- 系统托盘菜单根据语言配置显示对应文字
- CLI 输出保持英文不变
- MCP 工具描述保持英文不变

## 架构总览

```
config.yaml
  ui.language: zh | en (默认 zh)
        ↓
  Settings API (GET/PUT /api/settings)
        ↓
  ┌─────────────────┐    ┌──────────────────┐
  │   Web UI        │    │   系统托盘        │
  │  fetch + i18n   │    │  启动时读取 lang  │
  │  data-i18n 属性 │    │  重启后生效       │
  └─────────────────┘    └──────────────────┘
```

## 一、配置层

### config.yaml 新增字段

```yaml
ui:
  language: zh   # en 或 zh，默认 zh
```

### internal/config/config.go

`Config` 结构体新增：

```go
type UIConfig struct {
    Language string `yaml:"language"` // "en" or "zh"
}

type Config struct {
    // ... 已有字段
    UI UIConfig `yaml:"ui"`
}
```

**启动时语言初始化流程：**

`Load()` 读取 `config.yaml` 后，若 `UI.Language` 为空（字段缺失或首次启动）：

1. 调用 `detectSystemLanguage()` 检测系统语言：
   - 检查 `LANG` / `LC_ALL` 环境变量，含 `zh` 则返回 `"zh"`
   - macOS 额外检查 `defaults read NSGlobalDomain AppleLocale`
   - 其他情况返回 `"en"`
2. 将检测结果填入 `cfg.UI.Language`
3. 调用 `cfg.Save()` 将 `ui.language` **回写到 `config.yaml`**

后续所有组件（Web UI、托盘）统一读取 `config.yaml`，`config.yaml` 是唯一数据源。

```go
func detectSystemLanguage() string {
    for _, env := range []string{"LANG", "LC_ALL", "LANGUAGE"} {
        if v := os.Getenv(env); strings.HasPrefix(v, "zh") {
            return "zh"
        }
    }
    // macOS: check system locale
    if runtime.GOOS == "darwin" {
        out, err := exec.Command("defaults", "read", "NSGlobalDomain", "AppleLocale").Output()
        if err == nil && strings.HasPrefix(strings.TrimSpace(string(out)), "zh") {
            return "zh"
        }
    }
    return "en"
}
```

### Settings API 扩展

`GET /api/settings` 响应中增加 `ui.language` 字段。
`PUT /api/settings` 支持写入 `ui.language`，校验值为 `"en"` 或 `"zh"`。

## 二、Web UI 国际化

### 翻译字典结构

每个 HTML 文件底部内嵌统一的翻译脚本块：

```html
<script>
const I18N = {
  en: {
    "nav.search":        "Search",
    "nav.files":         "Files",
    "nav.status":        "Status",
    "nav.settings":      "Settings",
    // ... 完整英文字典
  },
  zh: {
    "nav.search":        "搜索",
    "nav.files":         "文件",
    "nav.status":        "状态",
    "nav.settings":      "设置",
    // ... 完整中文字典
  }
};

async function applyI18n() {
  let lang = "zh";
  try {
    const res = await fetch("/api/settings");
    const cfg = await res.json();
    lang = cfg?.ui?.language || "zh";
  } catch (_) {}
  const dict = I18N[lang] || I18N["zh"];
  document.querySelectorAll("[data-i18n]").forEach(el => {
    const key = el.getAttribute("data-i18n");
    if (dict[key]) el.textContent = dict[key];
  });
}

document.addEventListener("DOMContentLoaded", applyI18n);
</script>
```

### HTML 标记

所有需要翻译的文本元素加 `data-i18n` 属性，保留原始英文作为 fallback：

```html
<a href="/" data-i18n="nav.search">Search</a>
<button data-i18n="btn.reindex">Reindex</button>
<h2 data-i18n="section.results">Results</h2>
```

### settings.html 语言切换

settings 页面新增语言选择项：

```html
<label data-i18n="settings.language">Language</label>
<select id="lang-select">
  <option value="zh">中文</option>
  <option value="en">English</option>
</select>
<p class="hint" data-i18n="settings.language_hint">
  Tray menu language takes effect after restart.
</p>
```

保存时调用 `PUT /api/settings`，保存成功后 `location.reload()` 刷新页面立即生效。

## 三、系统托盘国际化

### internal/tray/tray.go

新增 `trayLabels` 结构体和 `labelsFor` 函数：

```go
type trayLabels struct {
    OpenDashboard string
    OpenKBDir     string
    Settings      string
    Quit          string
}

func labelsFor(lang string) trayLabels {
    if lang == "zh" {
        return trayLabels{
            OpenDashboard: "打开控制台",
            OpenKBDir:     "打开知识库目录",
            Settings:      "设置",
            Quit:          "退出",
        }
    }
    return trayLabels{
        OpenDashboard: "Open Dashboard",
        OpenKBDir:     "Open KB Directory",
        Settings:      "Settings",
        Quit:          "Quit",
    }
}
```

`Run(port int, kbPath string)` 改为 `Run(port int, kbPath string, lang string)`，调用 `labelsFor(lang)` 初始化菜单文字。

调用方（`main.go`）传入 `cfg.UI.Language`。

### 语言变更行为

- Web UI 语言切换：立即生效（`location.reload()`）
- 托盘语言切换：下次启动时生效，settings 页提示用户重启
